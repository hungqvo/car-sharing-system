# AdminAPI

AdminAPI api and webpage for car sharing system

## Configuration Edit the .env file with the database credential in order to connect to that database server ## Install dependency Use the package manager [pip](https://pip.pypa.io/en/stable/) to install all the dependency needed with the requirements.txt

`bash pip3 install -r requrements.txt `

## Usage

`python python3 main.py ### server is running ` Access the home page of the car sharing system admin at localhost:5000/
